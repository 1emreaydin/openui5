Get Ready

All you need to develop Web apps with OpenUI5 is a development environment and a browser. Choose between SAP Web IDE in the cloud for maximum convenience or set up your local development environment or maximum flexibility.

Both options are described in our documentation.

Start Coding

Try it out yourself - modify and extend this basic "Hello World!" example as you wish!

Now it is time to build your first OpenUI5 app! To start simple, we load OpenUI5 from our CDN and place UI5 controls into the HTML body, as you can see in the example below. An explanation of the code can be found after the code section. 

<html>
	<head>
		<title>Hello World</title>

		<script src='https://openui5.hana.ondemand.com/resources/sap-ui-core.js' 
   		id='sap-ui-bootstrap'
    		data-sap-ui-theme='sap_belize'
    		data-sap-ui-libs='sap.m'

		<script>
			var app = new sap.m.App("HelloWorldApp"); 
			var page = new sap.m.Page("HelloWorldPage", {
				title: "Hello World Page",
				content : new sap.m.Button({
					text : "Say Hello",
					press : function() {
						alert("Hello World!");	
					}
				})
			});
			app.addPage(page);
			app.placeAt('content');

		</script>
	</head>
		
		<body class='sapUiBody' id='content'>
	</body>
</html>

This is what the code above does:

The “script src” tag loads UI5 and sets the bootstrap ID, theme and main library.

Variables for new app, page and button produce the respective objects and set title, text and event handler. The “placeAt” tag inserts the button at the indicated position with given ID into the HTML document.
