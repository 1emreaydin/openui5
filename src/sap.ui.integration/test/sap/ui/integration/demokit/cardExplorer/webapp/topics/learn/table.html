Table Card

The table card is used to display multiple items in table view.

Properties

The table card contains an array of rows.

Table row properties

Property Type Required Description Version
columns array No Represents columns attributes 1.1.0
actions object No Represents actions that can be applied on the item 1.1.0

Table column properties

Property Type Required Description Version
title string No Represents language-dependent title of the column 1.1.0
width string No Defines the width of the column 1.1.0
value string No Represents text value of the column 1.1.0
icon icon No Represents column with icon 1.1.0
state string No Represents the state of the column 1.1.0
url string No Represents URL string 1.1.0
target string No Specifies where to open the 'url', if it is provided. 1.1.0
identifier string No ... 1.1.0
progressIndicator string No ... 1.1.0

Example

Defining the type and data for the card: 

{
	"sap.app": {
		"type": "card"
	},
	"sap.card": {
		"type": "Table",
		"header": {
			"title": "Sales Orders for Key Accounts",
			"subTitle": "Today",
			"status": {
				"text": "5 of 27"
			}
		},
		"content": {
			"data": {
				"request": {
					"url": "./cardcontent/tableitems.json"
				}
			},
			"row": {
				"columns": [
					{
						"label": "Sales Order",
						"value": "{salesOrder}",
						"identifier": true
					},
					{
						"label": "Customer",
						"value": "{name}"
					},
					{
						"label": "Net Amount",
						"value": "{netAmount}"
					},
					{
						"label": "Status",
						"value": "{status}",
						"state": "{statusState}"
					},
					{
						"label": "Avatar",
						"width": "12%",
						"icon": {
							"src": "{icon}"
						}
					},
					{
						"label": "Progress",
						"progressIndicator": {
							"percent": "{percent}",
							"text": "{percentValue}",
							"state": "{progressState}"
						}
					}
				]
			}
		}
	}
}

	<w:Card manifest="./manifest.json" width="400px" height="auto"/>
</mvc:View>
