Date and Time Formatters

The date formatter takes as a first parameter a valid JavaScript Date object, ISO 8601 date string or a valid date and time strings, and as a second parameter an object, which defines the format options. Underneath for the date formatter we use sap.ui.core.format.DateFormat and the second parameter options is passed unchanged to the getDateTimeInstance.

Param Type Description
date JS Date Any valid value, that can construct JavaScript Date object.
options object Formatting options.
Returns
string The formatted date.

The date parameter can come from different places including {{parameters.TODAY_ISO}}}, {{parameters.NOW_ISO}}}, data binding, Date.now() and many more.

Examples

A card with a date formatter using expression binding for today with {pattern: 'MMMM d, y'} as format options

"sap.card": {
	"type": "List",
	"header": {
		"title": "Order status {= format.date(Date.now(), {pattern: 'MMMM d, y'}) }"
	}
}
