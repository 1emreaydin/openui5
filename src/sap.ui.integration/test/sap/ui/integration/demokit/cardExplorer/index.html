Easy Card Integration in just two steps...

1

Declare a Cards manifest.json file

With the manifest.json you declare the cards header and content. You can use static data to be displayed or declare a request object for your data endpoint. Translations for static texts on the card, like the title and subtitle, can be provided in separate files. 
							{
  "sap.app": {
    "type": "card"
  },
  "sap.card": {
    "type": "List",
    "header": {
       "title": "Project Cloud Transformation",
       "subTitle": "Revenue"
    },
    "content": {
      "data": {
        "json": [{
          "Name": "Comfort Easy",
          "Description": "32 GB Digital Assistant",
          "Highlight": "Error"
        }]
      },
      "item": {
        "title": {
        "label": "{{title_label}}",
          "value": "{Name}"
        },
        "description": {
          "label": "{{description_label}}",
          "value": "{Description}"
        },
        "highlight": "{Highlight}"
      }
    }
  }
}
2

Consume the Card in your site

Insert a script tag refering the sap-ui-integration.js bundle to your site. This will initialize your page for Card usage.
Add a <ui-card> tag where the Card should appear.


  <script
	  src="https://ui5.sap.com/resources/sap-ui-integration.js"
	  id="sap-ui-bootstrap"
	  data-sap-ui-theme="sap_fiori_3"></script>
  <body>
    <ui-card
	    manifest="./cards/listsample.manifest.json">
    </ui-card>
  </body>

Done

Launch your page and see the result.
