Easy Card Integration in just two steps...

1

Define a Cards manifest.json file

With the manifest.json you define the cards header and content. You can use static data to be displayed or define a request object for your data endpoint. Translations for static texts on the card, like the title and subtitle, can be provided in separate files. 
{
  "sap.card": {
    "type": "Object",
    "header": {
      "icon": {
        "src": "../cardExplorer/images/Sabine_Mayer.png"
      },
      "title": "Donna Paulsen",
      "subTitle": "Executive Assistant"
    },
    "content": {
      "groups": [
        {
          "title": "Pearson Specter",
          "items": [
            {
              "label": "Mobile",
              "value": "+001 6101 34869-0",
              "type": "phone"
            },
            {
              "label": "Phone",
              "value": "+001 4231 5424 0231",
              "type": "phone"
            },
            {
              "label": "Email",
              "value": "donna@psl.com",
              "type": "email"
            }
          ]
        },
        {
          "title": "Legal Services Unit",
          "items": [
             {
              "label": "Direct Manager",
              "value": "Harvey Specter",
              "icon": {
                "src": "../cardExplorer/images/Laurent_Dubois.png"
                }
             },
             {
              "label": "Phone",
              "value": "+001 4242 4242 0873",
              "type": "phone"
             },
             {
              "label": "Email",
              "value": "thebeststcloser@psl.com",
              "type": "email"
             }
          ]
        }
      ]
    }
  }
}
2

Consume the Card in your site

Insert a script tag referring the sap-ui-integration.js bundle to your site. This will enable your page for card usage.
Add a <ui-card> tag where the card should appear.

<script
  src="https://ui5.sap.com/resources/sap-ui-integration.js"
  id="sap-ui-bootstrap"
  data-sap-ui-theme="sap_fiori_3"></script>
<body>
  <ui-card
    manifest="./cards/objectsample.manifest.json">
  </ui-card>
</body>

Done

Launch your page and see the result.
